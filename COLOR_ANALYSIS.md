- `#E8F0FF` - 매우 연한 파란색 (투명도 높음)
  - 홈 화면: 오늘의 수업 카드 그라데이션 시작점
  - 감정 차트: 배경
  - 갤러리: 작품 카드 색상
  
- `#D8E8FF` - 연한 파란색 (조금 더 진함)

  - 창작실: 두뇌 미니게임 카드 배경
  - 모달: 감정 모달 테두리
  - 갤러리: 아틀리에 장식 프레임
  
- `#C5D9FF` - 중간 파란색
  - 금주의 수업: 오늘 수업 카드 테두리/링
  - 홈 화면: 오늘의 수업 카드 테두리
  - 감정 차트: 막대 차트 테두리
  
- `#A8C8FF` - 더 진한 파란색
  - 창작실: 두뇌 미니게임 카드 테두리
  
- `#2A6CCF` - 진한 파란색 (텍스트/버튼용)

  - 버튼 배경
  - 아이콘 컨테이너 배경
  
- `#2560B8` - 더 진한 파란색 (hover/텍스트)

  - 텍스트 색상
  
- `#1F4F9E` - 매우 진한 파란색 (텍스트)

## 📋 문제점 요약

- **매우 연한**: `#E8F0FF`
- **연한**: `#D8E8FF`
- **중간**: `#C5D9FF`, `#A8C8FF`
- **진한**: `#2A6CCF`, `#2560B8`, `#1F4F9E`

**→ 통일 필요**: 배경용은 `#D8E8FF`로, 테두리는 `#A8C8FF`로, 강조/버튼은 `#2A6CCF` 계열로

**→ 통일 필요**: 배경은 `#DBF2E3`로, 테두리는 `#B5DFC7`로, Primary는 `#4C8F7E`로, 텍스트는 `#256A39`로

**→ 통일 필요**: 배경은 `#FFE5D4`로, 테두리는 `#FFC89C`로, Primary는 `#EB6A29`로
